# 一.项目面试:
    1.明确你的项目到底是做什么的,有哪些功能.
    2.明确你的项目的整体架构,在面试的时候能够清楚地画给面试官看并且清楚地指出从哪里调用到哪里、使用什么方式调用.
    3.明确你的模块在整个项目中所处的位置及作用.
    4.明确你的模块用到了哪些技术,更好一些的可以再了解一下整个项目用到了哪些技术
