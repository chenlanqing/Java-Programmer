# 一、Prometheus


修改默认端口9090：
- 修改yml配置文件：
```yml
scrape_configs:
  - job_name: "prometheus"
    static_configs:
      - targets: ["localhost:8090"]
```
- 按照启动prometheus
```
./prometheus --config.file=prometheus.yml --web.listen-address=:8090
```
