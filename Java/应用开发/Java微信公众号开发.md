# 一.开发环境的准备:
    1.微信公众号:
    2.外围映射工具(开发调试)
    与微信对接的URL要具备以下条件
    	(1).在公网上能够访问;
    	(2).端口只支持80端口
    	注:ngrok可以将内网地址映射到外网上,  http://www.tunnel.mobi/	
    	在CMD命令中先切换到ngrok所在的位置再进行如下操作
    	ngrok -config ngrok.cfg -subdomain example 8080
    	说明：
    	example-自己任意设置；
    	8080-tomcat的端口号
