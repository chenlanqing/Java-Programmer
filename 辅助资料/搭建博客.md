## 2、hexo常用命令

```sql
-- 安装hexo
npm install hexo-cli -g
-- 初始化
hexo init blog
-- 补充一些hexo博客的依赖
hexo install
-- 清除原来的缓存，它会直接把原来编译生成的静态文件夹public直接删除
hexo clean
-- 根据配置文件生成静态页面
hexo g
-- 启动服务器，默认4000端口
hexo s
-- 发布到git仓库
hexo d
```
